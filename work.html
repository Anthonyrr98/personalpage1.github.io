Core services

No two projects are the same, and each may include one, two or all of the following core services with appropriate options.

1
Design

Visual design

Web and product design and direction, from simple sites to complex systems. The latest methods strengthened by broad experience and an eye for detail.

May include...

  • Research & development
  • Pattern library / design system
  • Interactive prototypes
  • Design & accessibility audits
  • Pair-design
  • Coaching & critique
  • Original graphics & illustration
2
HTML, CSS, JS, PHP...

Front-end code

Efficient and future-proof, crafted with care and adherent to web standards. Progressive, accessible, responsive and device-tested by default.

May include...

  • Social integration & cards
  • Typographic scale
  • CSS pre-processing (Sass)
  • CSS animation / JS behaviour
  • Data capture with forms
  • Performance audit & budget
  • Version control with Git
3
Back-end

Content management

As simple as possible, as powerful as necessary; the right tool for the right job. Appropriate architecture, clear content strategy and clean data.

May include...

  • File-based CMS, e.g. Kirby
  • Static site generator, e.g. Eleventy
  • Database CMS, e.g. Craft
  • Dynamic social cards & sharing
  • Style guide / documentation
  • Data collection
  • Public and/or admin accounts

Other services include: consultation, mentoring, print design, writing, teaching, speaking, curation and event management.
