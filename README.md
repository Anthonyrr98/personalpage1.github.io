## 语法高亮使用<br/>
在head里嵌入语法高亮

```html
        <link rel="stylesheet" type="text/css" href="https://rlzhao.com/highlight/styles/monokai.min.css">
        <script src="https://rlzhao.com/highlight/highlight.min.js"></script>
        <script>
            hljs.highlightAll();
        </script>
```
